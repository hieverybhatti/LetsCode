Documentation-Dictionaries

Dictionaries are unordered collections of key:value pairs.
Values may have duplicates and can be of any data type, however keys have to be immutable types--string, number, or tuple--and must be unique.
You may create a dictionary by using {} braces or dic() function.

aDict = {1: "hello", 2: "bye"}
bDict = dict({1: "hello", 2: "bye"})

Accessing Elements

Unlike other data types that use index to access values, dictionary uses keys.
Keys can be used inside [] brackets or get() function.

print(aDict[1])
print(aDict.get(1))
Above print statements would both print "hello".

Adding and Removing Elements

New pairs can be added by defining key and value.

aDict = {1: "one", 2: "two", 3: "three"}
aDict[4] = "four"
Be aware that if the key already exists, doing so would update the value.

aDict[3] = "five"
print(aDict)
Since 3 already existed, the value would be updated and aDict would be {1: "one", 2: "two", 3: "five", 4: "four"}.
Items can also be removed by using pop() or del.

aDict = {1: "one", 2: "two", 3: "three"}
aDict[4] = "four"
aDict.pop(4)
del aDict[1]
Above code will update aDict to be {2: "two", 3: "three"}.
