Documentation-Strings

Strings are sequence of characters and can be created using single or double quotes.

a = "Hello"
b = 'Hello'
To assign multiline string to a variable, you can use """ or '''.

c = """The violets are blue,
the roses are red,
the violets are blue,
I love my loves."""
You can access characters in a string using index.

a = "Hello"
print(a[1])
The following code will print "e".
You can also get the length of a string with len() function.

a = "Goodbye!"
len(a)
You may concatenate two strings using + and repeat using *.

a = "Hello" + "John"
b = "Goodbye"
c = a + b * 2
a would be "HelloJohn" and c would be "HelloJohnGoodbyeGoodbye".

Similarly, you can slice a string into substrings by using :.

a = "HelloJohn"
b = a[0:4]
b would be "Hello".
