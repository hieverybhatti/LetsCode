Documentation-Recursion

Recursion is a type of method that calls itself. Remember that functions can call other functions. A function that calls itself is a recursive function.
The most popular example that uses recursion is factorial.

def factorial(x):
    if x == 1:
        return 1
    else:
        return (x * factorial(x - 1))
The following function will call itself until the number x is equal to 1. As you can see, every recursive function must have a non-recursive boundary case that defines a limiting condition. Once the condition is met, the recursion will stop.

Here is another example that uses recursive method: fibonacci sequence.

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return (fibonacci(n - 1) + fibonacci(n - 2))
