Functions are blocks of code that perform specific tasks.
Functions help organize the program, and they can receive and return data. Note that there are already built-in functions in Python, but addition to these, you can create your own function.

def function_name(parameters):
    codes
A function_name identifies the function, and parametersare optional in case there are values that need to be passed to the function. To call your defined function, you simply have to call the name, and if you want the function to return a value, you may do so with return.
def sayHi(name):
    greeting = "Hello " + name
    return greeting
print(sayHi("John")
The following code will print "Hello John".
