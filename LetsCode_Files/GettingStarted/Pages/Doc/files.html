Documentation-Files

Files store information in named locations on disk, and Python can handle files.
A file must be opened to be processed, and there are four modes for opening a file: r(read), w(write), a(append), or r+(read and write).

f = open("fileName.txt", "mode")
And after the file is processed, it must be closed using close() function.
f.close()

Reading a File

To read a file, the file must be opened in readable modes. Once it is opened, you can use read() function to read the whole file or readline() to read line by line. To read remaining lines in the file, you can also use readlines().

f = open("fileName.txt", "r")
for line in f:
    line = f.readline()
    line.rstrip("\n")
    ...
f.close()
Note that readline() reads "\n"(newline character) as well, so you may use rstrip() to remove it.

Writing and Appending a File

To write to a file, the file must be opened in writable modes. If the file is opened in w mode, any previous content will be overwritten. If it is opened in a mode, new text will be added at the end of existing content.

f = open("fileName.txt", "w")
f.write("Yes\n")
f.close()
f = open("fileName.txt", "a")
f.write("More Yes\n")
