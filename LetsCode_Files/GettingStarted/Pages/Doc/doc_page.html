Documentation-Loops

Loops are used to repeat certain part of the code.
There are two types of loops: for loops and while loops.

A for loop is generally used when the code repeats for a definite number of times.
A while loop is generally used when the code repeats until a condition is met.

Example:
for x in range(3):
  print("*")		 These two loops will both have same result with "***" but with different approach.
The for loop had a numeric range of repetition.
On the other hand, the while loop had a condition of y < 3 which was met after repeating the loop 3 times.
y = 0
while (y < 3):
  print("*")
  y += 1
